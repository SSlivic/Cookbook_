Recepti

Na računalniku mora biti instaliran java development kit(jdk).

Programa se zažene klikom na Recepti\dist\Recipes.jar

Programa se sastoji od dve opciji. Opcija unašanja recepta, in opcije upogleda v recepte, njihov odabir, update in refresh in printanje datega recepta.

Podatkovna baza kuvar.db se nahaja v istem fajlu Recepti\dist\kuvar.db je v obliki sqlite.

Izdelava programa je trajala tri dni.
